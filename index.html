Minimum Spanning Tree With Prim's Algorithim

This sketch is a visualization of a minimum spanning tree using my own implementation of Prim's Algorithim. A minimum spanning tree is the the shortest path to connect all vertices in a graph. Prim's Algorithim accomplishes this using the following steps. First it chooses any vertex as the origin. Then it chooses the closest vertex to the origin and adds it to the tree. After that it continually adds the next closest vertex to any other vertex already in the tree until all vertices are connected.

Click anywhere on the canvas to add a vertex.
